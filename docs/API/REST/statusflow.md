### REST API endpoint: statusflow

`http://IP-ADDRESS/statusflow`


Get actual process state

Format: `[HH:MM:SS] Actual process state`

Payload:
- No payload needed

Response:
- Content type: `HTML`
- Content: Query response
- Example: `[11:01:42] Idle - Waiting for Autostart`
