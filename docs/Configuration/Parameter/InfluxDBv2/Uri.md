# Parameter: URI

|                   | WebUI               | Config.ini
|:---               |:---                 |:----
| Parameter Name    | URI                 | Uri
| Default Value     | `http://IP-ADDRESS:PORT` | `http://IP-ADDRESS:PORT`


## Description

URI of the HTTP interface to InfluxDB, without trailing slash, e.g. `http://192.168.1.1:8086`.
