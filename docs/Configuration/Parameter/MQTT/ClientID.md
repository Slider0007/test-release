# Parameter: Client ID

|                   | WebUI               | REST API
|:---               |:---                 |:----
| Parameter Name    | Client ID           | clientid
| Default Value     | `watermeter`        | `watermeter`


## Description

Client ID used to connect to the MQTT broker<br>
Max length: 23 character (limit mentioned by MQTT specification [MQTT-3.1.3-5])
