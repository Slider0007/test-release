# Parameter: Log Files Retention

|                   | WebUI               | REST API
|:---               |:---                 |:----
| Parameter Name    | Log Files Retention | logfilesretention
| Default Value     | `5`                 | `5`
| Input Options     | `0` = no deletion<br> `1` .. &infin; | `0` = no deletion<br> `1` .. &infin;
| Unit              | Days                | Days


## Description

Number of days the log files are kept.
