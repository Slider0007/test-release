# Parameter: Data Files Retention

|                   | WebUI               | REST API
|:---               |:---                 |:----
| Parameter Name    | Data Files Retention | datafilesretention
| Default Value     | `30`                | `30`
| Input Options     | `0` = no deletion<br> `1` .. &infin; | `0` = no deletion<br> `1` .. &infin;
| Unit              | Days                | Days


## Description
